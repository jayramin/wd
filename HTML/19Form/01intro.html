Form Tag

Form Tag will helps to send client data to the server

for sending data to the server form has 2 methods

name attribute will helps to send client data to server sepratly

  1. GET
  2. POST

GET

  1. Get is Default method of form
  2. With the help of Get method we can send client data to the server using URL
  3. Get method sends its data by URL thats why its less secure
  4. Get method have limitations of 2048 bytes only
  5. Get method dnt support file uploading
  6. We can do bookmark of Get method data
  7. Get merthod is faster than post
  8. mainly we use get method for searching or sending one page data to other

POST

  1. With the help of Get method we can send client data to the server without using URL
  2. POST method sends its data without URL thats why its secure
  3. POST method doenst have any limitation
  4. POST method support file uploading
  5. We can't bookmark of POST method data
  6. POST is slower then get merthod
  7. mainly we use POST for sending data to the server while we need security and large amount of data

Form Attributes

Attribute Description
accept-charset Specifies the character encodings used for form submission accept-charset="utf-8"
action Specifies where to send the form-data when a form is submitted
autocomplete Specifies whether a form should have autocomplete on or off
enctype Specifies how the form-data should be encoded when submitting it to the server (only for method="post")
method Specifies the HTTP method to use when sending form-data
name Specifies the name of the form
novalidate Specifies that the form should not be validated when submitted
rel Specifies the relationship between a linked resource and the current document
target Specifies where to display the response that is received after submitting the form
