way to apply css

  1. Inline CSS [within tag with style attribute ] (not recommanded by w3c)
  2. Internal CSS [within page with style tag](not prefrabble for coding point of view)
  3. External CSS [exteranl file with .css extension and connect with HTML page by link tag](most prefrabble and adoptable)
